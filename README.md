## Estrutura de cada arquivo

Cada capítulo segue a mesma estrutura pedagógica:

1. **Objetivos de aprendizagem** — o que o aluno deve saber ao fim do capítulo
2. **Pré-requisitos** — conceitos necessários de capítulos anteriores
3. **Conteúdo** — seções com explicações, tabelas comparativas e exemplos de código
4. **Mapa conceitual** — visão geral das relações entre os tópicos
5. **Relevância para SI** — conexão com tecnologias e cenários reais de Sistemas de Informação
6. **Erros conceituais comuns** — equívocos frequentes e suas correções
7. **Questões de revisão** — exercícios para fixação
8. **Referência bibliográfica**
